sequenceDiagram
    participant Donor as Donor
    participant DonorUI as Donor UI
    participant System as System
    participant Inventory as Blood Inventory
    participant Database as Database
    
    Donor->>DonorUI: 1. Login
    DonorUI->>System: Authenticate
    System->>Database: Validate credentials
    Database-->>System: User data
    System-->>DonorUI: Auth token
    
    Donor->>DonorUI: 2. Check eligibility
    DonorUI->>System: Get donor profile
    System->>Database: Query donor & donations
    Database-->>System: Donor info + last donation
    System->>System: Calculate eligibility
    System-->>DonorUI: Eligibility status
    
    alt Donor is Eligible
        Donor->>DonorUI: 3. Book appointment
        DonorUI->>System: Request appointment slots
        System->>Database: Get available slots
        Database-->>System: Available appointments
        System-->>DonorUI: Display calendar
        
        Donor->>DonorUI: Select date & time
        DonorUI->>System: Confirm appointment
        System->>Database: Create appointment
        Database-->>System: Appointment ID
        System-->>DonorUI: Confirmation
        
        Donor->>DonorUI: 4. Donation day arrives
        DonorUI->>System: Check in
        System->>Database: Update appointment status
        Database-->>System: Status updated
        
        Donor->>DonorUI: 5. Record health metrics
        DonorUI->>System: Submit metrics (BP, Hb)
        System->>Database: Store health metrics
        Database-->>System: Metrics ID
        
        Donor->>DonorUI: 6. Donation completed
        DonorUI->>System: Complete donation
        System->>Database: Create donation record
        System->>Database: Calculate next eligible date
        System->>Inventory: Decrease blood count
        Inventory->>Database: Update inventory
        Database-->>System: Updates confirmed
        System-->>DonorUI: Donation recorded
    else Donor is Ineligible
        System-->>DonorUI: Display next eligible date
        DonorUI-->>Donor: Show countdown
    end
    
    Donor->>DonorUI: 7. View donation history
    DonorUI->>System: Get donation history
    System->>Database: Query donations
    Database-->>System: Donation records
    System-->>DonorUI: Display timeline
