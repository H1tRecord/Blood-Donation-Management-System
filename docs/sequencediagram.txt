sequenceDiagram
    participant User
    participant UI as App UI
    participant Sys as System Logic
    participant DB as Firebase DB

    == Authentication ==
    User->>UI: Login / Register
    UI->>DB: Validate & Fetch Profile
    DB-->>UI: Session & Role Redirect

    == Donor Workflow ==
    User->>UI: View Donor Dashboard
    UI->>Sys: Check Eligibility
    Sys->>DB: Fetch Last Donation
    DB-->>UI: Eligibility Status
    opt If Eligible
        User->>UI: Book Appointment
        UI->>DB: Save Appointment
        DB-->>UI: Confirmation
    end

    == Staff Workflow ==
    User->>UI: Manage Appointments & Inventory
    UI->>DB: Fetch Today's Data
    DB-->>UI: Summary Data
    User->>UI: Complete Donation
    UI->>Sys: Process Action
    Sys->>DB: Sync Database (History, Inventory)
    DB-->>UI: Acknowledgment

    == Admin Workflow ==
    User->>UI: Manage Accounts
    UI->>DB: Create / Update / Delete User
    DB-->>UI: Account Updated
